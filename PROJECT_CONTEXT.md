# Orinadus Project Context

This file is the generic repo-local context entry point for Orinadus.

For the deeper product-specific version, also read `ORINADUS_CONTEXT.md`.

## Purpose

Orinadus is the intelligence and research platform of the ecosystem.

It exists to monitor sources, detect signals, organize knowledge, support author desks, and publish structured insight.

## Boundary Summary

Orinadus owns intelligence gathering, signal handling, and authored interpretation.

It does not own orchestration infrastructure, persistent companion behavior, or creative tooling as primary system responsibilities.
